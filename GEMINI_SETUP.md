# DataHub - Настройка Gemini AI

Для использования ИИ-улучшения текста:

1. Получите API ключ: https://aistudio.google.com/apikey
2. Экспортируйте ключ:
   ```bash
   export GEMINI_API_KEY="ваш_ключ"
   ```
3. Перезапустите API сервер:
   ```bash
   python api_server.py
   ```

## Что делает ИИ:
- Очищает текст от навигации и мусора
- Переводит на русский язык если нужно
- Форматирует и улучшает читаемость
- Структурирует информацию
